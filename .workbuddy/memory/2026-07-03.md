## 权限管理弹窗修复（2026-07-03 上午 10:25）

### 用户反馈的问题
1. **点击"＋"按钮没反应**：不弹窗
2. **点击"复制权限"没反应**：不弹窗

### 根因分析
唯一ID方案（`_spm_` + 时间戳）有语法风险，导致某些浏览器解析失败。

### 修复方案（最终版）
**放弃唯一ID方案，改用 `querySelector()` 选择器**：
- `showConfirmModal`: 使用 `overlay.querySelector('.sd-confirm-ok')` 绑定事件
- `showPromptModal`: 使用 `overlay.querySelector('.sd-prompt-input')` 获取元素
- `showSelectModal`: 使用 `overlay.querySelector('.sd-prompt-input')` 获取元素
- **优点**：不需要唯一ID，代码简单可靠，不会出现ID冲突
